# Linux Files

```
/etc/shadow : local user's hashes

/etc/passwd : local user's

/etc/group : local groups

/etc/rc.d : startup services

/etc/init.d : service

/etc/hosts : known hostnames and IPs

/etc/hostname : full hostname with domain

/etc/network/intefaces : network configuration 

/etc/profile : system environment variables

/etc/apt/source.list : ubuntu/debian sources list

/etc/resolv.conf : nameserver configuration

/home/user/.bash_history : bash history of user

/usr/share/wireshark/manuf : vendor MAC lookup i.e. lists all the mac addresses with their vendors

~/.ssh : ssh keystore

/var/log : system log files (Linux)

/var/adm : system log files (Unix)

/var/spool/cron : list cron files(a directory)

/var/log/apache/access.log : apache connection log(a file)

/etc/fstab : static file system info(a file)
```
